# Dictionary App

A responsive Dictionary App built using HTML, CSS, and JavaScript. It allows users to search for English words and view their meanings, pronunciation, examples, and audio pronunciation using the Free Dictionary API.

## Features

- Search for English words
- View word meanings and examples
- Listen to pronunciation
- Responsive user interface
- Easy-to-use design

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Free Dictionary API
